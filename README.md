# DatumStudio MCP Core

Editor-first deterministic tooling foundation for Unity.

## Overview

DatumStudio MCP Core provides a minimal, opinionated foundation for building Unity editor tooling. This package serves as the canonical free core and base for paid tiers.

## Installation

### Unity Package Manager (Git URL)

1. Open Unity Package Manager (Window > Package Manager)
2. Click the `+` button
3. Select "Add package from git URL"
4. Enter: `https://github.com/DatumStudios/datumstudio-mcp-core.git?path=package/com.datumstudio.mcp.core`

Alternatively, add to your `Packages/manifest.json`:

```json
{
  "dependencies": {
    "com.datumstudio.mcp.core": "https://github.com/DatumStudios/datumstudio-mcp-core.git?path=package/com.datumstudio.mcp.core"
  }
}
```
