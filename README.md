# EditorMCP

Editor-first deterministic tooling foundation for Unity.

## Overview

EditorMCP provides a minimal, opinionated foundation for building Unity editor tooling. This package serves as the canonical free core and base for paid tiers.

## Installation

### Unity Package Manager (Git URL)

1. Open Unity Package Manager (Window > Package Manager)
2. Click the `+` button
3. Select "Add package from git URL"
4. Enter: `https://github.com/DatumStudios/EditorMCP.git?path=/package/com.datumstudios.editormcp#main`

Alternatively, add to your `Packages/manifest.json`:

```json
{
  "dependencies": {
    "com.datumstudios.editormcp": "https://github.com/DatumStudios/EditorMCP.git?path=/package/com.datumstudios.editormcp#main"
  }
}
```
