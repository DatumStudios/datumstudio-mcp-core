# Tests

This folder contains editor tests for DatumStudio MCP Core.

## Assembly Definition

The `DatumStudio.Mcp.Core.Tests.Editor.asmdef` assembly definition ensures tests only run in the Unity Editor and reference the appropriate assemblies.

## Running Tests

Tests can be run via:
- Unity Test Runner (Window > General > Test Runner)
- Command line with Unity's test runner

## Test Structure

Tests follow Unity's test framework conventions. See Unity's testing documentation for details.
