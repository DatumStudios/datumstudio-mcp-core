# Frequently Asked Questions

## General

### What is DatumStudio MCP Core?

DatumStudio MCP Core is an editor-first foundation package for Unity tooling. It provides the base infrastructure for DatumStudio's paid tier products.

### Is this free?

Yes, the core package is free and open source under Apache 2.0.

### What are the paid tiers?

Paid tiers provide additional functionality built on top of the core. See the main README for tier information.

## Technical

### Why is the Runtime folder empty?

The Runtime folder exists as a seam for potential future expansion. In v1, this package is editor-only.

### Can I use this in production?

Yes, but note that this is v0.1.0 - early stage software. Use at your own discretion.

### What Unity versions are supported?

See [Compatibility](./Compatibility.md) for detailed version support information.

## Support

### How do I get help?

Free tier users can open GitHub Issues. Paid tier users have access to tier-specific support channels.

### Will you add feature X?

Maybe. Open a GitHub Issue to discuss. We prioritize features that align with our philosophy and benefit the majority of users.

### Can I contribute?

Yes! See [CONTRIBUTING.md](../CONTRIBUTING.md) for guidelines.
